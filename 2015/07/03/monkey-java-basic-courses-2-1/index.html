##字符型(char)

char类型数据用来表示通常意义上的字符；注意以下几点：

  • 字符是由单引号包裹起来的单个字符；
  • Java字符使用unicode字符集；

unicode字符集是什么？

它为每种语言的每个字符设定了统一并唯一的二进制码。

详细参见WiKi百科Unicode词条

问题：为何ccc和c变量都是合法的呢？
回答：因为Java语言默认使用Unicode字符集，从而保证一个汉字和一个英文字母在计算机中占用的空间是一样的。

##布尔型(boolean)

boolean数据类型适用于逻辑运算，一般用于程序流程控制，在Java中只有2个值：

  • true : 表示“真”；
  • false : 表示“假”。
