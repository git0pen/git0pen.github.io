IO操作的目标

  • 从数据源当中读取数据，以及将数据写入到数据的目的地当中。

特点：

  • I/O系统的主要目标是为了对数据进行读写操作；
  • 数据的流向以Java程序为参照物；
  • I/O流可以有三种分类方法；
  • 有read和write两个主要方法。

IO的分类方法

第一种：

输入流；输出流。

第二种：

字节流；字符流。

第三种：

节点流；处理流。

特点：

  • BufferedReader：字符输入处理流；readline()方法，每次读取一行，返回一个字符串；
  • BufferedReader br = new BufferedReader(new FileReader(“xxx.txt”));

读取文件和写入文件的方法

核心类

  • InputStream和FileInputStream（从硬盘中文件读取数据）
  • OutputStream和FileOutputStream（把数据写入硬盘中文件）
