前言

在管理自己的代码的时候，我一直使用github。有的时候，私有项目不想公开，如果还要放到GitHub上，那就必须要花钱。解决这个问题的方法其实也很简单：在自己电脑上用虚拟机（VirtualBox）跑一个服务器（CentOS），配置好ssh和git server。正所谓：史上距离你最近的服务器。
