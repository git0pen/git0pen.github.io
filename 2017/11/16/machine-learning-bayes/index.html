  1. 公式：
    P(Y|X)=P(X|Y)P(Y)/P(X)
  2. 由联合概率公司推导而来：
    P(Y,X) = P(Y|X)P(X)=P(X|Y)P(Y)
  3. P(Y)叫做先验概率，P(Y|X)叫做后验概率，P(Y,X)叫做联合概率。
